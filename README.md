# Autodesign - AI Based Layout Generator

Autodesign is a simple layout design tool made by **King Nilesh**.  
You can upload images, generate layouts, and explore auto-design concepts.

## Features
- Upload image
- Auto-generate layout (demo)
- Simple clean UI
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open `index.html` in your browser
3. Enjoy the Autodesign demo
